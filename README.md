# finebearing (Industrial B2B Navbar Project)

This project implements a premium, modern "Liquid Glass" navigation bar for an industrial B2B website.

## Features implemented:
- Full-width edge-to-edge navbar
- Solid grey matte and liquid glass aesthetic
- Integrated glassy search bar with hover and focus effects
- Responsive navigation links (Product, About, Blog, Contact Us)
- Profile/Sign-in icon with tactile click animations
- Perfectly standardized 24px spacing and layout

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev` to start the development server
